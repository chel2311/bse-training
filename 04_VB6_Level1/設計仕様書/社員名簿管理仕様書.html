社員名簿管理アプリケーション 設計仕様書

1. 概要

社員情報の登録・削除・検索・一覧表示を行うWindowsデスクトップアプリケーション。
10年以上前に構築されたVB6アプリケーションの保守案件として管理する。

2. 開発環境

項目 仕様
言語 Visual Basic 6.0
UI VB6標準フォーム
一覧表示 Microsoft Windows Common Controls 6.0（ListView）
データ管理 インメモリ配列（将来的にDB対応予定）
IDE Visual Basic 6.0 IDE

4. 機能仕様

4.1 社員追加（FR-001）

  • 「追加」ボタン押下で入力欄の値を社員データとして追加する
  • 社員番号は重複不可。既に登録済みの社員番号の場合はエラー「この社員番号は既に登録されています」を表示する
  • 追加後、入力欄をクリアする
  • 追加後、一覧を更新して新しい社員が表示されること

4.2 社員削除（FR-002）

  • 一覧で行を選択した状態で「削除」ボタン押下
  • 削除前に「○○を削除しますか？」と確認ダイアログを表示する
  • 「はい」で削除実行、「いいえ」でキャンセル
  • 一覧が選択されていない場合は「社員を選択してください」と警告表示

4.3 検索（FR-003）

  • 氏名テキストボックスにキーワードを入力し「検索」ボタンで実行
  • 氏名に対して部分一致検索を行う
  • 検索結果が0件の場合「該当する社員はいません」と表示
  • キーワードが空の場合は全件表示

4.4 入力バリデーション（FR-004）

  • 社員番号: 必須。数字のみ許可。空欄の場合はエラー「社員番号を入力してください」、数字以外が含まれる場合はエラー「社員番号は数字で入力してください」
  • 氏名: 必須。空欄の場合はエラー「氏名を入力してください」
  • 入社日: IsDate関数を使用して日付形式チェックを行う。不正な場合はエラー「入社日を正しい日付形式(yyyy/mm/dd)で入力してください」

4.5 一覧表示（FR-005）

  • ListViewに以下のカラムを表示する:
  • 社員番号
  • 氏名
  • 部署
  • 入社日（yyyy/mm/dd形式）
  • 勤続年数（年単位）
  • フォーム読み込み時にサンプルデータを表示する

4.6 勤続年数計算（FR-006）

  • 入社日から現在日までの年数をDateDiff関数で計算する
  • 計算式: DateDiff("yyyy", 入社日, Now) で年数を取得
  • 一覧の勤続年数カラムに「○年」の形式で表示する

5. データ構造

5.1 Employee型（ユーザー定義型）

メンバー 型 説明
EmpNo String 社員番号
EmpName String 氏名
Department String 部署
HireDate Date 入社日

5.2 データ格納

  • モジュールレベルの動的配列 Employees() で管理
  • 社員数は EmployeeCount 変数で管理

6. 非機能要件

  • エラー発生時は適切なメッセージを表示し、アプリが異常終了しないこと
  • On Error GoTo によるエラーハンドリングを実装すること
  • Option Explicit を全モジュールで宣言すること
