```
  double avgAge = count > 0 ? (double)totalAge / count : 0;
  | 意味: 平均年齢を計算する
  | count > 0 ?: 件数が0より大きいかの条件分岐（三項演算子）
  | (double)totalAge / count: 合計を件数で割る（doubleにキャストして小数を保持）
  | : 0: 件数が0の場合は平均0とする（0で割るエラーの防止）
  | 仕様: FR-006「全顧客の年齢の平均（小数第1位まで）」に対応

  lblStatus.Text = $"件数: {count}件  合計年齢: {totalAge}  平均年齢: {avgAge:F1}";
  | 意味: ステータスラベルに結果を表示する
  | $"..." は文字列補間（{変数名}の部分が値に置き換わる）
  | {avgAge:F1}: avgAgeを小数第1位まで表示する書式指定
  | 例: 件数3件、合計105、平均35.0 → "件数: 3件  合計年齢: 105  平均年齢: 35.0"
```

## 補足: C# 基本文法の早見表

このコードで使われている主なC#文法を簡単にまとめる。

| 書き方 | 意味 | 例 |
|-------|------|-----|
| `if (条件) { ... }` | 条件が真の時だけ中の処理を実行 | `if (count == 0) { ... }` |
| `!` | 否定（真偽を反転） | `!int.TryParse(...)` = 変換失敗時にtrue |
| `return;` | メソッドの実行をそこで終了 | バリデーションエラー時の中断 |
| `var x = 値;` | 変数宣言（型は自動推論） | `var customer = new Customer(...)` |
| `new クラス名(...)` | オブジェクト（実体）を新規作成 | `new Customer(1, "田中", ...)` |
| `リスト.Add(要素)` | リストに要素を追加 | `customers.Add(customer)` |
| `リスト.Remove(要素)` | リストから要素を削除 | `customers.Remove(customer)` |
| `リスト.Find(条件)` | 条件に合う最初の要素を返す | `customers.Find(c => c.Id == 1)` |
| `リスト.FindAll(条件)` | 条件に合う全ての要素を返す | `customers.FindAll(c => c.Name.Contains("田"))` |
| `リスト.Count` | リストの要素数 | `customers.Count` → 3 |
| `リスト.Sum(c => c.プロパティ)` | プロパティの合計 | `customers.Sum(c => c.Age)` → 105 |
| `c => c.Id == 1` | ラムダ式（簡易的な関数の書き方） | 「cを受け取り、c.Id==1の結果を返す」 |
| `条件 ? 値A : 値B` | 三項演算子（条件分岐の省略形） | `count > 0 ? total/count : 0` |
| `?.` | null条件演算子（nullなら以降を実行しない） | `Value?.ToString()` |
| `??` | null合体演算子（左がnullなら右を使う） | `result ?? ""` |
| `$"文字列{変数}"` | 文字列補間（変数を埋め込む） | `$"件数: {count}件"` |
| `{値:F1}` | 書式指定（F1 = 小数第1位まで） | `{35.0:F1}` → "35.0" |
| `(int)値` | 型キャスト（型を変換） | `(int)row.Cells["Id"].Value` |
| `nextId++` | 後置インクリメント（使った後に+1） | ID=3を使い、nextIdが4になる |
| `+=` (イベント) | イベントハンドラの登録 | `btnAdd.Click += ... btnAdd_Click` |
| `string.IsNullOrWhiteSpace(s)` | 文字列がnull/空/空白のみかチェック | 入力バリデーションで使用 |
| `int.TryParse(s, out int n)` | 文字列→整数変換を試みる | 成功ならtrue、結果はnに格納 |
| `s.Contains(keyword, ...)` | 文字列にキーワードが含まれるか | 部分一致検索で使用 |
