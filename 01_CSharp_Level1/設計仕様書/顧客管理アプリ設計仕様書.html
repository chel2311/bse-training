顧客管理アプリケーション 設計仕様書

1. 概要

顧客情報の登録・表示・検索・削除を行うWindowsデスクトップアプリケーション。

2. 開発環境

項目 仕様
言語 C#
フレームワーク .NET Framework 4.8 / .NET 8
UI Windows Forms
DB SQL Server（将来対応。初期はインメモリ）
IDE Visual Studio 2022

4. 機能仕様

4.1 顧客追加（FR-001）

  • 「追加」ボタン押下で入力欄の値を顧客リストに追加する
  • IDは自動採番（既存の最大ID + 1）
  • 追加後、入力欄をクリアする
  • 追加後、一覧を更新して新しい顧客が表示されること
  • ステータスバーの件数・集計を更新する

4.2 顧客編集（FR-002）

  • 一覧で行を選択した状態で「編集」ボタン押下
  • 選択行の値が入力欄に表示される
  • 入力欄を修正後、再度「編集」ボタンで上書き保存
  • 一覧が選択されていない場合は「顧客を選択してください」と警告表示

4.3 顧客削除（FR-003）

  • 一覧で行を選択した状態で「削除」ボタン押下
  • 削除前に「○○を削除しますか？」と確認ダイアログを表示する
  • 「はい」で削除実行、「いいえ」でキャンセル
  • 一覧が選択されていない場合は「顧客を選択してください」と警告表示

4.4 検索（FR-004）

  • 検索テキストボックスにキーワードを入力し「検索」ボタンで実行
  • 氏名またはメールアドレスに対して部分一致検索を行う
  • 大文字・小文字を区別しない
  • 検索結果が0件の場合「該当する顧客はありません」と表示
  • 「全件表示」ボタンで検索をリセットし全データを表示

4.5 入力バリデーション（FR-005）

  • 氏名: 必須。空欄の場合はエラー「氏名を入力してください」
  • 電話番号: 任意。入力がある場合は数字とハイフンのみ許可
  • 年齢: 必須。0以上150以下の整数。範囲外はエラー「年齢は0～150の整数で入力してください」
  • メールアドレス: 任意。入力がある場合は「@」を含むこと

4.6 ステータス表示（FR-006）

  • 画面下部に以下を常時表示:
  • 現在の表示件数
  • 全顧客の年齢の合計
  • 全顧客の年齢の平均（小数第1位まで）
  • 追加・削除・検索時に自動更新される

5. データ構造

5.1 Customer クラス

プロパティ 型 説明
Id int 顧客ID（自動採番）
Name string 氏名
Phone string 電話番号
Age int 年齢

6. 非機能要件

  • エラー発生時は適切なメッセージを表示し、アプリがクラッシュしないこと
  • 一覧の選択行は青色でハイライトされること
