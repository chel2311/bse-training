在庫管理アプリケーション 設計仕様書

1. 概要

商品在庫の登録・編集・削除・検索・在庫金額管理を行うWindowsデスクトップアプリケーション。

2. 開発環境

項目 仕様
言語 VB.NET
フレームワーク .NET 8
UI Windows Forms
DB なし（インメモリ管理）
IDE Visual Studio 2022

4. 機能仕様

4.1 商品追加（FR-001）

  • 「追加」ボタン押下で入力欄の値を在庫リストに追加する
  • 商品IDは自動採番（既存の最大ID + 1）
  • 追加後、入力欄をクリアする
  • 追加後、一覧を更新して新しい商品が表示されること
  • ステータスバーの件数・在庫金額合計を更新する

4.2 商品編集（FR-002）

  • 一覧で行を選択した状態で「編集」ボタン押下
  • 選択行の値が入力欄に表示される
  • 入力欄を修正後、再度「編集」ボタンで上書き保存
  • 一覧が選択されていない場合は「商品を選択してください」と警告表示

4.3 商品削除（FR-003）

  • 一覧で行を選択した状態で「削除」ボタン押下
  • 削除前に「○○を削除しますか？」と確認ダイアログを表示する
  • 「はい」で削除実行、「いいえ」でキャンセル
  • 一覧が選択されていない場合は「商品を選択してください」と警告表示

4.4 在庫金額計算（FR-004）

  • 在庫金額 = 在庫数 × 単価
  • 一覧の「在庫金額」列に自動計算結果を表示する
  • 商品の追加・編集時に自動で再計算される

4.5 検索（FR-005）

  • 検索テキストボックスにキーワードを入力し「検索」ボタンで実行
  • 商品名に対して部分一致検索を行う
  • 大文字・小文字を区別しない
  • 検索結果が0件の場合「該当する商品はありません」と表示
  • 検索テキストが空の場合は全件表示

4.6 ステータス表示（FR-006）

  • 画面下部に以下を常時表示:
  • 現在の表示商品数
  • 全商品の在庫金額合計
  • 追加・編集・削除・検索時に自動更新される

4.7 入力バリデーション（FR-007）

  • 商品名: 必須。空欄の場合はエラー「商品名を入力してください」
  • 在庫数: 必須。0以上の整数。整数でない場合はエラー「在庫数は0以上の整数で入力してください」
  • 単価: 必須。1以上の整数。整数でない場合または1未満の場合はエラー「単価は1以上の整数で入力してください」

4.8 在庫警告表示（FR-008）

  • 在庫数が5以下の商品は、DataGridViewの該当行を黄色（Color.Yellow）で表示する
  • 在庫数が6以上の行は通常表示（デフォルト色）
  • 一覧更新時に毎回判定を行う

5. データ構造

5.1 Product クラス

プロパティ 型 説明
Id Integer 商品ID（自動採番）
ProductName String 商品名
Category String カテゴリ（食品/飲料/日用品）
Stock Integer 在庫数
UnitPrice Integer 単価
StockValue Integer（ReadOnly） 在庫金額（在庫数 × 単価の計算結果）

6. 非機能要件

  • エラー発生時は適切なメッセージを表示し、アプリがクラッシュしないこと
  • 一覧の選択行は青色でハイライトされること
  • 在庫数が5以下の行は黄色で警告表示されること
