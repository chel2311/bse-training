# 図形描画ツール（CAD基礎モジュール） 設計仕様書

## 1. 概要

CADソフトウェアの基盤となる図形計算モジュール。図形の生成・管理・面積/周長計算を行うC++ライブラリ。
GUIは対象外。コンソールアプリケーションとしてロジックの検証を行う。

## 2. 開発環境

| 項目 | 仕様 |
|------|------|
| 言語 | C++ (C++17) |
| コンパイラ | MSVC (Visual Studio 2022) |
| プラットフォーム | Windows x64 |
| UI | コンソール（標準入出力） |
| IDE | Visual Studio 2022 |

## 3. クラス設計

### 3.1 クラス継承図

```
Shape（基底クラス・抽象クラス）
  ├── Circle（円）
  ├── Rectangle（矩形）
  └── Triangle（三角形）

ShapeManager（図形管理クラス）
  └── std::vector<Shape*> shapes  ← Shapeポインタのリスト
```

### 3.2 Shape クラス（基底クラス）

| メンバ | 種別 | 型 | 説明 |
|--------|------|-----|------|
| type | protected | std::string | 図形タイプ名 |
| Shape() | コンストラクタ | - | 初期化 |
| virtual ~Shape() | デストラクタ | - | 仮想デストラクタ（派生クラスの適切な破棄のため） |
| GetArea() | 純粋仮想関数 | double | **図形の面積を返す** |
| GetPerimeter() | 純粋仮想関数 | double | **図形の周長を返す** |
| GetInfo() | 純粋仮想関数 | std::string | 図形の情報文字列を返す |
| GetType() | メンバ関数 | std::string | 図形タイプ名を返す |

### 3.3 Circle クラス（円）

| メンバ | 種別 | 型 | 説明 |
|--------|------|-----|------|
| radius | private | double | 半径 |
| Circle(double radius) | コンストラクタ | - | 半径を指定して生成 |

**計算式:**
- **面積 = pi x r x r**（pi = 3.14159265358979）
- **周長 = 2 x pi x r**

**バリデーション:**
- **半径（radius）は正の値（> 0）であること**
- 不正な値の場合は `std::invalid_argument` 例外を送出

### 3.4 Rectangle クラス（矩形）

| メンバ | 種別 | 型 | 説明 |
|--------|------|-----|------|
| width | private | double | 幅 |
| height | private | double | 高さ |
| Rectangle(double width, double height) | コンストラクタ | - | 幅と高さを指定して生成 |

**計算式:**
- **面積 = width x height**
- **周長 = 2 x (width + height)**

**バリデーション:**
- **幅（width）、高さ（height）はともに正の値（> 0）であること**
- 不正な値の場合は `std::invalid_argument` 例外を送出

### 3.5 Triangle クラス（三角形）

| メンバ | 種別 | 型 | 説明 |
|--------|------|-----|------|
| base_len | private | double | 底辺の長さ |
| height | private | double | 高さ |
| side_a | private | double | 辺a |
| side_b | private | double | 辺b |
| side_c | private | double | 辺c |
| Triangle(double base, double height, double a, double b, double c) | コンストラクタ | - | 底辺、高さ、3辺を指定して生成 |

**計算式:**
- **面積 = base_len x height / 2**
- **周長 = a + b + c**

**バリデーション:**
- **底辺（base_len）、高さ（height）、3辺（a, b, c）はすべて正の値（> 0）であること**
- 不正な値の場合は `std::invalid_argument` 例外を送出

### 3.6 ShapeManager クラス（図形管理）

| メンバ | 種別 | 型 | 説明 |
|--------|------|-----|------|
| shapes | private | std::vector\<Shape*\> | 図形ポインタのリスト |
| ShapeManager() | コンストラクタ | - | 初期化 |
| ~ShapeManager() | デストラクタ | - | **保持する全図形のメモリを解放（delete）** |
| AddShape(Shape* shape) | メンバ関数 | void | 図形を追加 |
| RemoveShape(int index) | メンバ関数 | bool | 指定インデックスの図形を削除（**deleteしてからリストから除去**） |
| GetTotalArea() | メンバ関数 | double | **全図形の面積合計を返す** |
| GetShapeCount() | メンバ関数 | int | 図形の総数を返す |
| FindByType(std::string type) | メンバ関数 | std::vector\<Shape*\> | **指定タイプの図形リストを返す** |
| PrintAll() | メンバ関数 | void | 全図形の情報を標準出力に表示 |

## 4. 機能仕様

### 4.1 図形の追加（FR-001）
- `ShapeManager::AddShape` で図形ポインタを管理リストに追加する
- 追加する図形は `new` で生成されたヒープオブジェクトであること

### 4.2 図形の削除（FR-002）
- `ShapeManager::RemoveShape` で指定インデックスの図形を削除する
- **削除時は対象図形の `delete` を行い、メモリを解放すること**
- インデックスが範囲外の場合は `false` を返す

### 4.3 面積合計の計算（FR-003）
- `ShapeManager::GetTotalArea` で全図形の面積の合計を返す
- **各図形の `GetArea()` を呼び出し、その合計値を返す**
- 図形が0件の場合は `0.0` を返す

### 4.4 図形タイプ検索（FR-004）
- `ShapeManager::FindByType` で指定タイプ名に一致する図形のリストを返す
- タイプ名は "Circle", "Rectangle", "Triangle" のいずれか

### 4.5 入力バリデーション（FR-005）
- **全ての寸法値（半径、幅、高さ、辺の長さ）は正の値（> 0）であること**
- 不正な値の場合はコンストラクタで `std::invalid_argument` 例外を送出
- エラーメッセージには不正なパラメータ名を含めること

### 4.6 メモリ管理（FR-006）
- **`new` で生成した図形オブジェクトは、`ShapeManager` のデストラクタで確実に `delete` すること**
- `ShapeManager` のスコープ終了時に保持する全図形のメモリが解放されること
- メモリリークが発生しないこと

## 5. 非機能要件

- 計算精度: `double` 型を使用し、円周率は `3.14159265358979` を定数として定義すること
- 例外発生時にアプリケーションがクラッシュしないよう、`main` 関数で適切にキャッチすること
- すべてのクラスでヘッダファイル（.h）とソースファイル（.cpp）を分離すること
